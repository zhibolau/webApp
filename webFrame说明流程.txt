http ---> WSGI server ---> WSGI 函数



设计web框架接口：
    1. URL路由： 用于URL 到 处理函数的映射     webFrame 做 url 到函数的映射 
    2. URL拦截： 用于根据URL做权限检测
    3. 视图： 用于HTML页面生成
    4. 数据模型： 用于抽取数据（见models模块）
    5. 事物数据：request数据和response数据的封装（thread local）


全局ThreadLocal对象:   实现事物数据接口, 实现request 数据和response数据的存储
